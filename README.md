## Setup
- ایجاد محیط مجازی پایتون

```commandline
python -m venv .venv
```
- فعال کردن محیط مجازی

```commandline
.venv\Scripts\activate
```
- نصب کتاب خانه های مورد نیاز

```commandline
pip install -r requirements.txt
```
- اجرای برنامه
```commandline
App.py
```
